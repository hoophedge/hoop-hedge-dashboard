# HoopHedge Dashboard

Public-facing dashboard for the HoopHedge automated trading signal system.

Hosted via GitHub Pages. Stats are auto-updated by HoopHedge after each scan via the GitHub API.

## Files

- `index.html` — Dashboard page
- `stats.json` — Performance metrics (auto-updated by HoopHedge)

## Setup

1. Enable GitHub Pages in repo Settings → Pages → Source: main branch
2. Add your GitHub PAT to HoopHedge's `config.json`
3. Stats update automatically after each scan
